# MAS Secure Messenger

Основні мови та стек: **React** і **TypeScript**. Проєкт розбитий на веб, десктоп та мобайл, із гарячим оновленням у дев-середовищі для швидкої перевірки перед складанням `exe`.

## Структура

- `apps/web` — браузерний клієнт (Vite + React + TS).
- `apps/desktop` — десктопний клієнт (Electron + React + TS).
- `apps/mobile` — мобільний клієнт (Expo + React Native + TS).
- `packages/shared` — спільні типи та утиліти.

## Дев-режим з гарячим оновленням

- Сервер: `npm run dev:server`
- Веб: `npm run dev:web`
- Десктоп: `npm run dev:desktop`
- Мобайл: `npm run dev:mobile`

В усіх клієнтах увімкнено hot reload / fast refresh, що дозволяє перевіряти зміни без повної збірки.

## Перевірка перед `exe`

- `npm run typecheck` — базова перевірка типів.
- `npm run build:desktop` — збірка десктопного клієнта.

## SMS авторизація (dev)

За замовчуванням сервер працює у dev-режимі й повертає `devCode` у відповіді `POST /auth/request`.
Це дозволяє локально протестувати SMS-логін без зовнішнього провайдера.

## Дизайн

Інтерфейс побудований з акцентом на чисту типографіку, м’які градієнти, картки та прев’ю чату, щоб демонструвати “красивий” сучасний вигляд.
